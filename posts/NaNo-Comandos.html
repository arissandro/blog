NaNo Comandos

Publicado em 12 de Junho de 2025

Categoria: Linux, Ferramentas, Terminal

Vamos direto ao assunto:
Se você está procurando um manual bem explicado sobre como usar os comandos do editor Nano, você está no lugar certo.

O símbolo ^ significa Ctrl
O símbolo M significa Alt ou ESC.

Abaixo, uma lista completa de comandos úteis para o dia a dia no editor Nano:

Comando Descrição
M-6 Copiar Texto (Copy Text)
^K Cortar Texto (Cut Text)
^U Colar Texto (Uncut Text)
^O Salvar o Arquivo
^X Sair (Exit)
^W Procurar Texto (Where Is)
^A Mover Para o Início da Linha (Move To Beginning of Line)
^E Mover Para o Final da Linha (Move to End of Line)
^Y Move a Página Para Cima (Move One Page Up)
^V Move a Página Para Baixo (Move One Page Down)
^F Move Cursor Um Caracter à Frente (Forward One Character)
^B Retorna Cursor Um Caracter (Backward One Character)
^M Inserir Nova Linha (Insert Newline)
^R Inserir Arquivo (Read File)
^J Justificar Parágrafo (Justify)
^T Verificar Erro de Ortografia (Spell Check)
^C Exibir Informações do Cursor (Current Cursor Position)
^G Exibir Ajuda (Help)
^L Redesenha a Tela (Refresh Screen)
^D Apaga Caracter à Direita do Cursor (Delete Character)
^H Exibe Ajuda

Essa lista é para você que está aprendendo a usar o terminal. O editor Nano é excelente para quem quer aprender a trabalhar sem interface gráfica, como no Ubuntu Server, por exemplo. De Nada XD.

Essa Lista Foi Testada no Nano Versão 7.2
